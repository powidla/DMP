# DMP
This repository contains utility code to generate, process and analyze data from [FriendOrFoe](https://github.com/powidla/Friend-Or-Foe) repo.

Specifically, $\texttt{data generator}$ contains code to generate pairwise metabolic interactions, $\texttt{processing}$ cleans and converts data for the Friend or Foe formatting style.
